FEATURE DETAILS
===============

Timeframe Parameter:
  Query Parameter: timeframe
  Default Value: 1h
  Valid Values: 15m, 1h, 4h, 1d
  Validation: Regex ^(15m|1h|4h|1d)$
  Backwards Compatible: Yes (default applies)

Extended EMA Response:
  New Fields:
    - ema10: 10-period EMA
    - ema20: 20-period EMA
    - ema50: 50-period EMA
    - price_above_ema10: boolean
    - ema10_above_ema20: boolean
    - ema20_above_ema50: boolean
    - ema50_above_ema200: boolean
    - trend_hierarchy_score: 0-4 (bullish alignment strength)

Consensus & Confidence:
  Consensus: BULLISH, BEARISH, or NEUTRAL
  Confidence: 0.0-1.0 (14 indicators weighted)

EXAMPLE REQUESTS
================

Test 1h (Default):
  curl "http://localhost:8116/api/v1/indicators/BTC%2FUSDT:USDT/current" \
    -H "Authorization: Bearer TOKEN"

Test 15m:
  curl "http://localhost:8116/api/v1/indicators/BTC%2FUSDT:USDT/current?timeframe=15m" \
    -H "Authorization: Bearer TOKEN"

Test 4h:
  curl "http://localhost:8116/api/v1/indicators/BTC%2FUSDT:USDT/current?timeframe=4h" \
    -H "Authorization: Bearer TOKEN"

Test 1d:
  curl "http://localhost:8116/api/v1/indicators/BTC%2FUSDT:USDT/current?timeframe=1d" \
    -H "Authorization: Bearer TOKEN"

Test Error (Invalid Timeframe):
  curl "http://localhost:8116/api/v1/indicators/BTC%2FUSDT:USDT/current?timeframe=5m" \
    -H "Authorization: Bearer TOKEN"
  (Returns: 422 Unprocessable Entity)
